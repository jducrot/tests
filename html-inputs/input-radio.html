Testing radio buttons

Published: February 08, 2020

Radio button group with no initially checked radio

results
  • Firefox allows Tab key to focus each radio button until a selection is made.
  • Safari, Chrome, and Edge (legacy and chromium) allow Tab key to go to the first or last radio button, depending on which direction users are navigating the DOM, until a selection is made.

Radio button group with an initially checked radio

results
All browsers treat a radio group with a pre-checked radio button as a single focus stop.
