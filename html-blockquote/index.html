blockquote element tests

Published: August 3, 2019

Note: navigate by blockquotes with NVDA by use of Q button. CTRL+INSERT+Q to open JAWS dialog of blockquotes.

Blockquote with no content

results

AX ignores empty blockquote.

FF: IA2 ignores empty blockquote. MSAA accRole: "grouping".

Chromium: MSAA accRole: 'BLOCKQUOTE', IA2 role: section, UIA: UIA_Custom

FF/Chrome + NVDA: ignored

FF/Chrome + JAWS: ignored

Blockquote with no content, but with cite attribute

results

AX ignores blockquote.

FF IA2 ignores empty blockquote. MSAA accRole: "grouping".

Chromium: MSAA accRole: 'BLOCKQUOTE', IA2 role: section, UIA: UIA_Custom

FF/Chrome + NVDA: ignored

FF/Chrome + JAWS: ignored

Blockquote with no content, but with title and cite attributes

results

AX ignores blockquote.

FF IA2 exposes no role. MSAA accRole: "grouping". title exposed as accName.

Chromium: MSAA accRole: 'BLOCKQUOTE', IA2 role: section, UIA: UIA_Custom. All Name from title.

FF/Chrome + NVDA: ignored

FF/Chrome + JAWS: ignored

Blockquote with no content, but aria-label and cite attribute

results

AX exposes as group named by aria-label but exposes no special info for cite attribute.

FF IA2 exposes no role. MSAA accRole: "grouping". aria-label exposed as accName.

Chromium: MSAA accRole: 'BLOCKQUOTE', IA2 role: section, UIA: UIA_Custom. All Name from aria-label.

FF/Chrome + NVDA: ignored

FF/Chrome + JAWS: ignored

Blockquote with paragraph element

Text within a paragraph.

results

AX ignores blockquote.

FF IA2 exposes no role. MSAA accRole: "grouping". No accName.

FF/Chrome + NVDA: announced as "blockquote".

FF/Chrome + JAWS: announced as "blockquote".

Blockquote with aria-label and with paragraph element

Text within a paragraph.

results

AX exposes as group named by aria-label.

FF IA2 exposes no role. MSAA accRole: "grouping". aria-label exposed as accName.

FF/Chrome + NVDA: announced as "blockquote" with accName.

FF/Chrome + JAWS: announced as "blockquote". accName exposed in blockquote dialog.

Blockquote with title and with paragraph element

Text within a paragraph.

results

AX exposes as group named by title.

FF IA2 exposes no role. MSAA accRole: "grouping". title exposed as accName.

FF/Chrome + NVDA: announced as "blockquote" with accName.

FF/Chrome + JAWS: announced as "blockquote". accName exposed in blockquote dialog.

Blockquote with title and aria-label and with paragraph element

Text within a paragraph.

results

AX exposes as group named by aria-label, title exposed as "help" text.

FF IA2 exposes no role. MSAA accRole: "grouping". aria-label exposed as accName. title exposed as description.

FF/Chrome + NVDA: announced as "blockquote" with accName. Description only announced if navigating by hot key.

FF/Chrome + JAWS: announced as "blockquote". accName exposed in blockquote dialog. Description not announced.

Blockquote with title, cite attr and aria-label and with paragraph element

User agents may allow users to follow such citation links, but they are primarily intended for private use...

results

AX exposes as group named by aria-label, title exposed as "help" text. No exposure of cite attribute.

FF IA2 exposes no role. MSAA accRole: "grouping". aria-label exposed as accName. title exposed as description.

FF/Chrome + NVDA: announced as "blockquote" with accName. Description only announced if navigating by hot key.

FF/Chrome + JAWS: announced as "blockquote". accName exposed in blockquote dialog. Description not announced.

Chrome/IE11 + JAWS will announce cited URL when VC navigates to blockquote.

Blockquote with paragraph element, with cite attribute

results

FF/Chrome + NVDA: announced as "blockquote".

FF/Chrome + JAWS: announced as "blockquote". JAWS dialog exposes cite attribute's URL as identifier.

Chrome/IE11 + JAWS will announce cited URL when VC navigates to blockquote.
