results

  • macOS VoiceOver can navigate to all hidden content.

    Hidden links and buttons can be activated.

    Hidden text within link is part of link's accName.

  • iOS VoiceOver can navigate to all hidden content.

    Hidden links and buttons can be activated.

    Hidden text within link is NOT part of link's accName.

  • TalkBack + Chrome can only find focusable hidden content.

    Hidden links and buttons can be activated.

    Hidden text within link is part of link's accName.

  • TalkBack + Firefox do not find any hidden content.

    Hidden text within link is NOT part of link's accName.

  • NVDA + Chrome can only find focusable hidden content.

    Hidden links and buttons can be activated.

    Hidden text within link is NOT part of link's accName.

  • NVDA + Firefox do not find any hidden content.

    Hidden text within link is NOT part of link's accName.

Cannot access any hidden interactive elements with Tab key or with a screen reader not enabled.
