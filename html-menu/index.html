menu with no children

results
  • macOS VoiceOver w/Safari or Chrome do not expose this pattern as a list
  • Completely ignored, no menu AX role / reported as ignored by Safari dev tools
  • macOS Firefox accessibility dev panel exposes it as a "text container"
  • macOS Chrome accessibility panel notes: "Element not interesting for accessibility."
